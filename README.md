# 🦕 SitePaleo

Site de notícias sobre **paleontologia** desenvolvido com Node.js e EJS, criado como projeto da cadeira de programação do curso Técnico em Informática.

---

## 📋 Sobre o projeto

O SitePaleo é um portal de notícias temático focado em descobertas e curiosidades paleontológicas. O backend é construído com **Node.js + Express**, e as páginas são renderizadas pelo motor de templates **EJS**, permitindo conteúdo dinâmico no servidor antes de entregar ao navegador.

---

## 🛠️ Tecnologias utilizadas

| Tecnologia | Uso |
|---|---|
| Node.js | Runtime do servidor |
| Express | Framework web e roteamento |
| EJS | Motor de templates (HTML dinâmico) |
| CSS | Estilização das páginas |
| JavaScript | Interatividade no cliente |

---

## 🗂️ Estrutura do projeto

```
SitePaleo/
├── bin/            # Inicialização do servidor
├── public/         # Arquivos estáticos (CSS, imagens, JS client-side)
├── routes/         # Definição das rotas Express
├── views/          # Templates EJS das páginas
├── app.js          # Configuração principal do Express
├── package.json    # Dependências do projeto
└── .gitignore
```

## 🚀 Como rodar localmente

### Pré-requisitos

- [Node.js](https://nodejs.org/) instalado na máquina

### Passos

**1. Clone o repositório**
```bash
git clone https://github.com/ZionSants/SitePaleo.git
cd SitePaleo
```

**2. Instale as dependências**
```bash
npm install
```

**3. Inicie o servidor**
```bash
npm start
```

**4. Acesse no navegador**
```
http://localhost:3000
```
